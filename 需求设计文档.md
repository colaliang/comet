## UI/UX设计规范

### 设计原则
- **简洁明了**: 突出核心价值，减少认知负担
- **信任感**: 专业的视觉设计建立用户信任
- **引导性**: 清晰的视觉层次引导用户行动
- **现代感**: 符合当前设计趋势的现代化界面

### 色彩方案
```css
/* 主色调 */
--primary: #6366f1;     /* 靛蓝色 - 科技感 */
--secondary: #8b5cf6;   /* 紫色 - AI感 */
--accent: #06b6d4;      /* 青色 - 活力感 */

/* 中性色 */
--gray-50: #f9fafb;
--gray-900: #111827;

/* 状态色 */
--success: #10b981;
--warning: #f59e0b;
--error: #ef4444;
```

### 字体系统
- 标题：Inter字体，字重600-700
- 正文：Inter字体，字重400-500
- 中文：Noto Sans SC，保证中文显示效果

### 组件设计
- **按钮**: 圆角8px，渐变背景，悬停效果
- **卡片**: 阴影效果，圆角12px，边框淡化
- **表单**: 清晰的标签，错误状态提示
- **导航**: 固定顶部，透明背景，滚动时变实
